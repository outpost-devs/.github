## 🛠️ Current Project: Drive-Shift

**Drive-Shift** is our inaugural utility application designed to migrate files and folders directly server-to-server between two Google Drive accounts. 

### Why it exists:
When a Google Drive runs out of space, it halts Gmail sync. The standard workaround is to download everything locally and re-upload it to a fresh account. Drive-Shift transfers folders and files directly within Google's backend infrastructure—eliminating the local device bottleneck.

* **Repository**: [drive-shift](https://github.com/outpost-devs/drive-shift)
* **Tech Stack**: Next.js, Java Spring Boot 3, Supabase (PostgreSQL), Tailwind CSS, Docker, Render, Vercel.
